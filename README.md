# GITHUB_STATE_OF_MIND

CLI + GUI (PyQt6) para auditar, analizar y optimizar repositorios de GitHub en masa usando la API de GitHub.

## Quickstart

```bash
export GITHUB_TOKEN="..."
python gh_manager.py estado --user <owner> --output table
```
